## HCL Domino 2026（版本 14.5.1）正式發布

HCL 於 2026 年 3 月 19 日正式發布了 HCL Domino 2026（版本 14.5.1），為 Notes 客戶端、Domino 伺服器和開發人員帶來了多項新功能和改進。

### 主要新功能

- **Domino IQ**：引入了內建的人工智慧功能，允許在本地運行大型語言模型，並提供 Java 和 LotusScript 類別，以及與 OpenAI 相容的 REST API 支援。

- **AutoUpdate**：增強了自動更新功能，現在可用於安裝 Windows 和 Linux 平台的軟體更新，包括主要版本。

- **OIDC 支援**：Domino 現在可作為 OIDC 提供者，並可強制執行僅使用 Passkey 的身份驗證。

### Notes 客戶端的改進

- **全新安裝程式**：重新設計的 Windows 安裝程式，模組直接放置在檔案系統中，類似於 Mac 的快速安裝過程。

- **用戶界面改進**：引入了新的預設字體（Inter），並在郵件、日曆、對話框和模板中進行了多項 UI 清理。

- **工作區改進**：更清晰的圖示大小，提升了用戶體驗。

### 伺服器端的增強

- **性能提升**：增強了伺服器性能，提供更強大的安全性和更好的管理/配置工具。

- **XPages 支援多檔案上傳**：使應用程式更具靈活性。

### Traveler 14.5.1 的新功能

- **線上會議支援**：擴展了與 HCL Verse Mobile 的線上會議支援，包括同步可用的線上會議服務列表、自動連結線上會議到新日曆條目的設定，以及直接從移動客戶端更新線上會議數據的能力。
