## HCL Domino 2026（版本 14.5.1）正式發布

HCL 於 2026 年 3 月 19 日正式發布了 HCL Domino 2026（版本 14.5.1），此版本帶來多項新功能與改進，旨在提升用戶體驗與系統效能。

### 主要新功能與改進

- **Domino IQ**：
  - 引入內建的人工智慧功能，支援在本地運行大型語言模型，並提供 Java 和 LotusScript 類別，以及與 OpenAI 相容的 REST API 支援。

- **AutoUpdate**：
  - 增強的自動更新功能，現在可用於安裝軟體更新，包括主要版本，簡化了系統維護流程。

- **OIDC 支援**：
  - Domino 現在可作為 OpenID Connect（OIDC）提供者，並可強制執行僅使用 Passkey 的身份驗證，提升安全性。

- **開發環境更新**：
  - 更新至最新版本的 Eclipse（4.30）和 Java（21 LTS），為開發者提供更現代化的開發環境。

- **LotusScript 改進**：
  - 引入串流支援，提升程式語言的靈活性與效能。
