歡迎！這是 **Domino News** 的第一篇文章。

本站專注於 [HCL Domino](https://www.hcl-software.com/domino) 與其周邊生態系（Notes、Domino REST API、Volt MX、Nomad、AppDev Pack、Sametime 等）的技術深度文章與生態系動態。

## 為什麼要做這個站

HCL Domino 的生態系資訊很分散：HCL 官方、HCL Ambassador 部落格、planetlotus.org 聚合、OpenNTF 專案、GitHub release notes、開發者社群論壇 …… 中文圈尤其少。本站每天發一篇繁中 + 英文文章，主軸是 HCL 官方類別／功能的深度教學，當有值得整理的新訊時則發布新聞型整理，帶來生態系最新動態。

## 預期主題

- 後端類別深掘（LotusScript / @Formula / Java / SSJS）
- 安全性與管理（TLS、NRPC encryption、ACL、OIDC 等）
- 產品版本更新與重大新功能
- AI 與 Domino 整合（Domino IQ）
- 容器化、REST API 與社群擴充工具

本站的程式碼跟發文流程公開於 [GitHub](https://github.com/bryanHsiao/domino-news)，歡迎 issue 或 PR。
