歡迎！這是 **Domino News** 的第一篇文章。

本站專注於 [HCL Domino](https://www.hcl-software.com/domino) 與其周邊生態系（Notes、Domino REST API、Volt MX、Nomad、AppDev Pack、Sametime 等）的技術深度文章與生態系動態。

## 為什麼要做這個站

HCL Domino 的生態系資訊很分散：HCL 官方、HCL Ambassador 部落格、planetlotus.org 聚合、OpenNTF 專案、GitHub release notes、開發者社群論壇 …… 中文圈尤其少。本站每天發一篇繁中 + 英文文章，主軸是 HCL 官方類別／功能的深度教學，當有值得整理的新訊時則發布新聞型整理，帶來生態系最新動態。

## 題材怎麼來

目前題材以 HCL 官方文檔的類別／功能深度教學為主（例如 [DQL 入門](/domino-news/posts/dql-getting-started)、[NotesViewNavigator 入門](/domino-news/posts/notes-view-navigator)、[NotesView.GetAllDocumentsByKey 全攻略](/domino-news/posts/getalldocumentsbykey)），用 [OpenNTF/ls-classmap](https://github.com/OpenNTF/ls-classmap) 當座標、追蹤哪些 class 還沒寫過。當有 HCL 官方公告、HCL Ambassador 部落格新訊、OpenNTF 動態值得整理時，會改成當日新聞型整理。

## 品質承諾

- 來源優先順序：HCL 官方頁 > HCL Ambassador / OpenNTF > 知名業務夥伴部落格（panagenda、prominic 等）
- 不引用無法驗證的內容；每篇文章末尾都會列出實際參考的來源連結
- 嚴重事實錯誤無法修正時，當天寧可不發，也不發有錯的內容

## 預期主題

- 產品版本更新（release notes 整理）
- Volt MX、Nomad、Domino REST API 的開發實務與教學
- LotusScript / @Formula / Java 後端類別深掘
- 安全性與管理（admin、ID vault、HTTP 標頭、OIDC）
- 社群活動與技術分享
- AI 與 Domino 整合趨勢（如 Domino IQ）

## 整套工作流程開源

包含產文流程、失敗草稿的救援機制等，[全部公開於 GitHub](https://github.com/bryanHsiao/domino-news)。歡迎參考、回報問題、或提出整套架構的改進建議。
