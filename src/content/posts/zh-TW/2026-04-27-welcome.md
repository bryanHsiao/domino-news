歡迎！這是 **Domino News** 的第一篇文章。

本站專注於 [HCL Domino](https://www.hcl-software.com/domino) 與其周邊生態系（Notes、Domino REST API、Volt MX、Nomad、AppDev Pack、Sametime 等）的最新動態。

## 運作方式

每天由 GitHub Actions 排程觸發 OpenAI（含 Web Search），整理過去 24-72 小時內的相關新聞與部落格文章，自動產出繁體中文與英文兩種版本，並標上對應的 tag。

- 文章來源優先選擇官方頻道、HCL Ambassador 部落格、社群（planetlotus.org、openntf.org）與 GitHub release notes
- 不引用無法驗證的內容
- 每篇文章皆附上實際參考的來源連結

## 預期主題

- 產品更新與 release notes
- Volt MX、Nomad、Domino REST API 的開發實務
- 社群活動與技術分享
- AI 與 Domino 整合趨勢
