## HCL Domino 2026 (Version 14.5.1) Officially Released

On March 19, 2026, HCL officially released HCL Domino 2026 (version 14.5.1), bringing a host of new features and improvements aimed at enhancing user experience and system performance.

### Key Features and Enhancements

- **Domino IQ**:
  - Introduces built-in artificial intelligence capabilities, supporting the local execution of large language models and providing Java and LotusScript classes, along with OpenAI-compatible REST API support.

- **AutoUpdate**:
  - Enhanced AutoUpdate feature now supports installing software updates, including major releases, simplifying system maintenance processes.

- **OIDC Support**:
  - Domino can now function as an OpenID Connect (OIDC) provider and enforce Passkey-only authentication, bolstering security measures.

- **Development Environment Updates**:
  - Updated to the latest versions of Eclipse (4.30) and Java (21 LTS), offering developers a more modern development environment.

- **LotusScript Improvements**:
  - Introduces streaming support, enhancing the flexibility and performance of the programming language.
