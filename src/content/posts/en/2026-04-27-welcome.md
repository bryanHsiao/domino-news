Welcome to **Domino News** — our first post.

This site focuses on [HCL Domino](https://www.hcl-software.com/domino) and its broader ecosystem: Notes, Domino REST API, Volt MX, Nomad, AppDev Pack, Sametime, and more.

## How it works

A GitHub Actions cron job triggers OpenAI (with Web Search) every day to curate news and blog posts from the last 24–72 hours, generating both Traditional Chinese and English versions with relevant tags.

- Sources prioritise official HCL channels, HCL Ambassador blogs, community sites (planetlotus.org, openntf.org), and GitHub release notes
- No unverifiable content is cited
- Every article links to the actual sources consulted

## What to expect

- Product updates and release notes
- Hands-on development with Volt MX, Nomad, and the Domino REST API
- Community events and technical knowledge sharing
- Trends in AI + Domino integration

The first fully automated post is scheduled to land tomorrow.
