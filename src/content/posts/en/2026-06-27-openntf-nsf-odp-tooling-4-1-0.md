On March 28, 2026, OpenNTF announced the release of NSF ODP Tooling version 4.1.0, aimed at enhancing the development experience for HCL Domino developers.

## Key Updates

- **JavaSourceClassLoader Improvements**: This release modifies the JavaSourceClassLoader to utilize ConcurrentHashMaps, enhancing performance and addressing [Issue #319](https://github.com/OpenNTF/org.openntf.nsfodp/pull/321).

- **Log Message Level Adjustments**: Several INFO-level log messages in NSFODPContainer have been lowered to DEBUG level, resolving [Issue #320](https://github.com/OpenNTF/org.openntf.nsfodp/pull/322).

## Download and Additional Information

Developers can download NSF ODP Tooling 4.1.0 from the [OpenNTF official website](https://openntf.org/main.nsf/project.xsp?r=project%2FNSF+ODP+Tooling%2Freleases%2FD1AE627FE95698F786258B0B0056CACD) and view the full changelog.

NSF ODP Tooling is an open-source project designed to provide more efficient development tools for HCL Domino developers. For more information about the project, please visit the [NSF ODP Tooling project page](https://openntf.org/main.nsf/project.xsp?r=project%2FNSF+ODP+Tooling%2Fsummary).
