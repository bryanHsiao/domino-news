On March 10, 2026, HCL officially released HCL Nomad 1.0.19, bringing several improvements and fixes to Nomad for web browsers and Domino servers. Users can download this version from the [My HCLSoftware Portal](https://developer.ds.hcl-software.com/t/hcl-nomad-for-web-browsers-and-hcl-nomad-server-on-domino-1-0-19-now-officially-released/172443).

## Key Updates

### User Interface Enhancements

This release updates several controls to align with the HCL Enchanted design language, including:

- **Rich Text Lite**: Provides a more intuitive text editing experience.
- **Time and Time Zone Controls**: Improves usability for time selection and time zone settings.
- **Color Picker and Color Field**: Offers a more user-friendly color selection tool.
- **Link Hotspot**: Enhances link management capabilities.

### Bug Fixes

This version addresses multiple known issues, enhancing system stability and performance. For a detailed list of fixes, refer to the [HCL Nomad User Documentation](https://help.hcl-software.com/nomad/1.0_web/PDF/nomad_web.pdf).

## Download and Installation

Users can download HCL Nomad 1.0.19 from the [My HCLSoftware Portal](https://developer.ds.hcl-software.com/t/hcl-nomad-for-web-browsers-and-hcl-nomad-server-on-domino-1-0-19-now-officially-released/172443). It is recommended to back up existing systems before upgrading and consult the official documentation to ensure a smooth installation process.
