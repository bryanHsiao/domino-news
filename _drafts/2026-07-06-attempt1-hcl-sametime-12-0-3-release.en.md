HCL Sametime 12.0.3 brings several new features and enhancements to real-time communication and online meetings, aiming to improve user experience and productivity. Here are the key updates:

## Chat Read Receipts

Users can now see when their messages have been read by recipients, reducing uncertainty and enabling more efficient, responsive conversations. ([help.hcl-software.com](https://help.hcl-software.com/sametime/v1203/admin/whatsnew1203.html?utm_source=openai))

## Mobile Chat Search

Quickly locate specific messages or conversations from your mobile device using keywords and filters. This feature makes it easy to retrieve important information without wasting time scrolling through chat history. ([help.hcl-software.com](https://help.hcl-software.com/sametime/v1203/admin/whatsnew1203.html?utm_source=openai))

## Meeting Recording Transcripts

Meeting recordings can now be automatically transcribed, providing a searchable, written record of the discussion. This helps teams stay aligned, follow up on action items, and ensure no key points are missed. ([help.hcl-software.com](https://help.hcl-software.com/sametime/v1203/admin/whatsnew1203.html?utm_source=openai))

## Mobile Auto Status & Persistent Notifications

User status on mobile devices will now automatically update based on activity, even through server session timeouts. Persistent notifications help users stay informed and maintain accurate availability, minimizing disruptions. ([help.hcl-software.com](https://help.hcl-software.com/sametime/v1203/admin/whatsnew1203.html?utm_source=openai))

## Improved Guest User Experience

Joining a meeting as a guest is now faster and easier, with fewer steps and a more intuitive interface. These improvements help reduce friction for external participants and support more seamless collaboration. ([help.hcl-software.com](https://help.hcl-software.com/sametime/v1203/admin/whatsnew1203.html?utm_source=openai))

## Premium Chat Features Now in Domino CCB

File transfer, screen capture, and instant meeting link sharing are now included in the standard Domino Complete Collaboration Bundle (CCB) entitlement—at no extra cost—enhancing chat functionality for all users. ([help.hcl-software.com](https://help.hcl-software.com/sametime/v1203/admin/whatsnew1203.html?utm_source=openai))

## Chat and Meetings Accessibility

Sametime is now aligned with accessibility requirements, offering screen reader support, keyboard navigation, high-contrast display options, and accessible meeting controls to support all users across web and mobile. ([help.hcl-software.com](https://help.hcl-software.com/sametime/v1203/admin/whatsnew1203.html?utm_source=openai))

## Enchanted and React Design Updates

The updated user interface now features the latest HCL Enchanted and React design systems, delivering a modern, consistent experience that aligns with other HCL Digital Solutions products for better usability and visual continuity. ([help.hcl-software.com](https://help.hcl-software.com/sametime/v1203/admin/whatsnew1203.html?utm_source=openai))

These updates in HCL Sametime 12.0.3 aim to make communication more efficient, inclusive, and integrated, enabling teams to collaborate with confidence, wherever they are. ([help.hcl-software.com](https://help.hcl-software.com/sametime/v1203/admin/whatsnew1203.html?utm_source=openai))

## Download and Installation Instructions

HCL Sametime 12.0.3 is available for download at [My HCLSoftware](https://my.hcltechsw.com/). For additional information, refer to the [HCL Sametime Administration Guide 12.0.3](https://help.hcl-software.com/sametime/v1203/admin/index.html).
