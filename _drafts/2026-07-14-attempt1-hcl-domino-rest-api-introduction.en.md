## What is the HCL Domino REST API?

The HCL Domino REST API provides a secure RESTful interface that allows developers to access HCL Domino servers and databases using various programming languages. This enables developers to leverage modern development tools and frameworks to extend and modernize existing Domino applications. ([opensource.hcltechsw.com](https://opensource.hcltechsw.com/Domino-rest-api/topicguides/introducingrestapi.html?utm_source=openai))

## Key Features

- **Security**: The Domino REST API inherits the security model of Notes and Domino, utilizing JSON Web Token (JWT) and Scopes for authentication, ensuring secure data access. ([opensource.hcltechsw.com](https://opensource.hcltechsw.com/Domino-rest-api/topicguides/introducingrestapi.html?utm_source=openai))

- **Standardization**: The API is based on OpenAPI 3.0 specifications, providing interactive documentation that facilitates understanding and usage for developers. ([opensource.hcltechsw.com](https://opensource.hcltechsw.com/Domino-rest-api/references/openapidefinitions.html?utm_source=openai))

- **Extensibility**: The API's architecture supports multiple API versions and can be extended to add more functionalities as needed. ([opensource.hcltechsw.com](https://opensource.hcltechsw.com/Domino-rest-api/references/openapidefinitions.html?utm_source=openai))

## Installation and Configuration

1. **Download**: Obtain the latest Domino REST API installer or Docker image from the HCL Software Portal. ([opensource.hcltechsw.com](https://opensource.hcltechsw.com/Domino-rest-api/tutorial/quickstart.html?utm_source=openai))

2. **Install**: Follow the installation guide to set up and start the REST API on a Domino server or Notes client. ([opensource.hcltechsw.com](https://opensource.hcltechsw.com/Domino-rest-api/tutorial/quickstart.html?utm_source=openai))

3. **Configure**: Use the Admin UI or API to create schemas and scopes for your databases, defining accessible design elements and operations. ([opensource.hcltechsw.com](https://opensource.hcltechsw.com/Domino-rest-api/tutorial/quickstart.html?utm_source=openai))

## Use Cases

- **Web Applications**: Display job openings stored in a Domino database on the company website without requiring Java or JavaScript skills. ([hcl-software.com](https://www.hcl-software.com/blog/domino/new-domino-rest-apis-are-now-available?utm_source=openai))

- **Mobile Applications**: Utilize Volt MX to create a front-end mobile app for travel approval systems running on Domino, allowing employees to submit requests and managers to approve them. ([hcl-software.com](https://www.hcl-software.com/blog/domino/new-domino-rest-apis-are-now-available?utm_source=openai))

- **Application Integration**: Integrate contracts and customer data stored in Domino with Salesforce CRM, providing sales representatives with a comprehensive view of necessary information. ([hcl-software.com](https://www.hcl-software.com/blog/domino/new-domino-rest-apis-are-now-available?utm_source=openai))

## Conclusion

The HCL Domino REST API offers developers a powerful and secure tool to access and extend Domino applications using modern programming languages and frameworks. By adhering to standardized API specifications and inheriting Domino's security model, developers can confidently bring existing Domino applications into contemporary development environments.
