On March 10, 2026, HCL officially released HCL Nomad for web browsers and HCL Nomad Server on Domino version 1.0.19, now available for download on the [My HCLSoftware Portal](https://developer.ds.hcl-software.com/t/hcl-nomad-for-web-browsers-and-hcl-nomad-server-on-domino-1-0-19-now-officially-released/172443).

## Key Updates

### User Interface Enhancements

This release includes updates to several controls to align with the HCL Enchanted design language, such as:

- **Rich Text Lite**: Provides a more intuitive text editing experience.
- **Time and Time Zone Controls**: Improved usability for time selection.
- **Color Picker and Color Field**: Offers a more user-friendly color selection tool.
- **Link Hotspot**: Enhanced link management capabilities.

### Bug Fixes

The update also addresses various known issues, enhancing system stability and performance. For a detailed list of fixes, refer to the [HCL Nomad Product Documentation](https://help.hcl-software.com/nomad/1.0_web/nomad_web_new.html?hl=language%2Csupport).

## Download and Installation

Users can download HCL Nomad 1.0.19 from the [My HCLSoftware Portal](https://developer.ds.hcl-software.com/t/hcl-nomad-for-web-browsers-and-hcl-nomad-server-on-domino-1-0-19-now-officially-released/172443). It is recommended to back up existing configurations before upgrading and to follow the official installation guidelines to ensure a smooth upgrade process.

## Compatibility Considerations

When upgrading to HCL Nomad 1.0.19, ensure that your HCL Domino server is running version 14.5.1 or later to maintain optimal compatibility and performance. More information on compatibility can be found in the [HCL Nomad Product Documentation](https://help.hcl-software.com/nomad/1.0_web/nomad_web_new.html?hl=language%2Csupport).

## Conclusion

The release of HCL Nomad 1.0.19 brings significant UI enhancements and bug fixes, improving the overall user experience. Users are encouraged to upgrade to this version to take advantage of the latest features and improvements.
