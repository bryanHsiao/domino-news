HCL Domino 2026 (version 14.5.1) was officially released on March 19, 2026, bringing a host of new features and improvements across the Notes client, Domino server, and for developers.

## New Features in the Notes Client

- **Redesigned Windows Installer**: The installation process has been overhauled, with modules placed directly in the file system, similar to the streamlined Mac installation process.

- **User Interface Enhancements**: A smoother UI, improved workspace and calendar functionalities, and support for OpenID Connect (OIDC) for easier, more secure logins.

## Enhancements in the Domino Server

- **Performance and Security**: Enhanced performance, stronger security measures, and improved administrative and configuration tools.

- **XPages Multi-File Upload Support**: XPages now supports multi-file uploads, making applications more flexible.

## New Features in Traveler 14.5.1

- **Online Meeting Support**: In conjunction with HCL Verse Mobile, Traveler 14.5.1 supports online meeting functionalities, including synchronization of available online meeting services and settings for automatically linking online meetings to new calendar entries.

For more detailed information, refer to the official announcement [HCL Domino 2026 (14.5.1) is here!](https://developer.ds.hcl-software.com/t/hcl-domino-2026-14-5-1-is-here/172452) and the feature overview [HCL Notes/Domino 14.5.1 is Here – All New Features at a Glance](https://www.madicon.de/blog/posts-blog/hcl-notesdomino-1451-is-here-all-new-features-at-a-glance/).
