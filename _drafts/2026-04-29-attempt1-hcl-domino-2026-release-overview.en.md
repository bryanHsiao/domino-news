HCL Domino 2026 (version 14.5.1) was officially released on March 19, 2026, bringing a host of new features and improvements for Notes users, Domino server administrators, and developers.

## New Features in the Notes Client

- **Redesigned Windows Installer**: The installation process has been overhauled, with modules placed directly in the file system, similar to the streamlined Mac installation process. ([madicon.de](https://www.madicon.de/blog/posts-blog/hcl-notesdomino-1451-is-here-all-new-features-at-a-glance/?utm_source=openai))

- **User Interface Enhancements**: A new default font (Inter) has been introduced, along with UI cleanups across Mail, Calendar, dialogs, and templates, enhancing the overall visual experience. ([panagenda.com](https://www.panagenda.com/blog/hcl-notes-2026-sneak-peek/?utm_source=openai))

- **Calendar Improvements**: Enhanced calendar functionalities provide a more intuitive event management and visualization experience. ([panagenda.com](https://www.panagenda.com/blog/hcl-notes-2026-sneak-peek/?utm_source=openai))

## Enhancements in the Domino Server

- **Performance and Security Upgrades**: The server has been optimized for better performance and fortified with stronger security features to ensure system stability and data protection. ([developer.ds.hcl-software.com](https://developer.ds.hcl-software.com/t/hcl-domino-2026-14-5-1-is-here/172452?utm_source=openai))

- **Improved Admin and Configuration Tools**: Administrative and configuration tools have been refined to simplify server management processes. ([developer.ds.hcl-software.com](https://developer.ds.hcl-software.com/t/hcl-domino-2026-14-5-1-is-here/172452?utm_source=openai))

- **Multi-File Upload Support in XPages**: XPages now supports multi-file uploads, increasing application flexibility. ([developer.ds.hcl-software.com](https://developer.ds.hcl-software.com/t/hcl-domino-2026-14-5-1-is-here/172452?utm_source=openai))

## New Features in Traveler

- **Enhanced Synchronization Reliability**: Improvements in synchronization ensure users remain productive on mobile devices. ([developer.ds.hcl-software.com](https://developer.ds.hcl-software.com/t/hcl-domino-2026-14-5-1-is-here/172452?utm_source=openai))

- **Online Meeting Support**: In conjunction with HCL Verse Mobile, Traveler now supports online meetings, including synchronization of available online meeting services, automatic linking of online meetings to new calendar entries, and the ability to update online meeting data directly from the mobile client. ([madicon.de](https://www.madicon.de/blog/posts-blog/hcl-traveler-1451-new-features-and-technical-changes/?utm_source=openai))

These new features and enhancements in HCL Domino 2026 aim to elevate user experience, bolster system performance and security, and provide more flexible application development tools.
