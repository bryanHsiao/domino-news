HCL Nomad 1.0.19 has been officially released, bringing several enhancements and bug fixes to improve the user experience.

## Key Updates

### User Interface Enhancements

In this release, the following controls have been refreshed to align with the HCL Enchanted design language:

- Rich text lite
- Time and time zone controls
- Color picker and color field
- Link hotspot

These updates modernize the interface, enhancing usability and aesthetics.

### Bug Fixes

1. **Fixed Error When Saving Documents**:

   - **Issue**: Users encountered the error message "Field length stored in document is incorrect" when saving documents.
   - **Solution**: This release addresses the issue, ensuring documents save without errors.

2. **Resolved Duplicate Server Entries in Open Application Dialog**:

   - **Issue**: The Open Application dialog's server list displayed the user's home mail server multiple times.
   - **Solution**: The problem has been fixed, ensuring the server list displays unique entries.
