## Overview of Recent HCL Nomad iOS Updates

HCL Nomad is a client that enables users to access Domino applications directly from their mobile devices. Recent iOS versions have introduced several new features and improvements to enhance user experience and performance.

### Version 1.0.55

- **Support for iPadOS 26 System Menu Bar**: On iPad (iPadOS 26 and later), the "Create" and "Actions" menus are now available in the new system menu bar.
- **Continuous Background Replication**: Active replication now continues for up to 30 seconds after Nomad enters the background.

### Version 1.0.53

- **Updated Dialog List Button**: The Dialog List button has an updated icon and improved size.
- **Updated "Open Application" Dialog**: The "Up one level" button icon has been updated.
- **About Page Information Update**: The About page now includes the version's year and month with the product name.

### Version 1.0.52

- **Rich Text Lite Field Update**: Updated to use the HCL Enchanted design language.
- **New "Inter" Font**: The "Inter" font is now included and set as the default sans-serif font for most locales.

### Version 1.0.51

- **Multiple Control Updates**: Radio buttons, checkboxes, list boxes, combo boxes, text field separators, and tabbed tables have been updated to use the HCL Enchanted design language.

### Version 1.0.50

- **Support for iPadOS 26 Menu Bar and Window Controls**.

### Version 1.0.48

- **Improved Tab Bar on Tablets**: Tabs are now grouped by application and can be scrolled horizontally. A new shortcut opens the main application tab when a document link is opened from an external source.
- **Open Replicas from Replication Page**: Replicas can now be opened by double-tapping a replica entry on the replication page.
- **LotusScript Access to C API**.
- **Improved Styling of Recently Used Apps Page**.

These updates aim to enhance user experience and provide more stable application operation. For more details, refer to the [HCL Nomad iOS Release Notes](https://help.hcl-software.com/nomad/1.0/hcln_whatsnew.html?scLang=de).
