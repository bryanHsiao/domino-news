HCL Domino 2026（版本 14.5.1）於 2026 年 3 月 19 日正式發布，為 Notes 客戶端、Domino 伺服器和開發者帶來了多項新功能和改進。

## Notes 客戶端的新功能

- **全新 Windows 安裝程式**：重新設計的安裝程式，模組直接放置在檔案系統中，類似於 Mac 的快速安裝過程。 ([madicon.de](https://www.madicon.de/blog/posts-blog/hcl-notesdomino-1451-is-here-all-new-features-at-a-glance/?utm_source=openai))

- **用戶介面改進**：更流暢的用戶介面，改進的工作區和日曆功能，提升了整體使用體驗。 ([developer.ds.hcl-software.com](https://developer.ds.hcl-software.com/t/hcl-domino-2026-14-5-1-is-here/172452?utm_source=openai))

## Domino 伺服器的增強功能

- **性能提升**：增強的性能和更強的安全性，提供更穩定的伺服器運行環境。 ([developer.ds.hcl-software.com](https://developer.ds.hcl-software.com/t/hcl-domino-2026-14-5-1-is-here/172452?utm_source=openai))

- **管理工具改進**：改進的管理和配置工具，使管理員能夠更有效地管理伺服器。 ([developer.ds.hcl-software.com](https://developer.ds.hcl-software.com/t/hcl-domino-2026-14-5-1-is-here/172452?utm_source=openai))

- **XPages 的多檔案上傳支持**：XPages 現在支持多檔案上傳，增加了應用程序的靈活性。 ([developer.ds.hcl-software.com](https://developer.ds.hcl-software.com/t/hcl-domino-2026-14-5-1-is-here/172452?utm_source=openai))

## 開發者的更新

- **Domino REST API 的改進**：最新版本的 Domino REST API 提供了新的功能和改進，增強了開發者的能力。 ([opensource.hcltechsw.com](https://opensource.hcltechsw.com/Domino-rest-api/whatsnew/index.html?utm_source=openai))

- **Domino IQ 的引入**：引入了安全的 AI 功能（Domino IQ），提供了本地運行大型語言模型的能力，並提供了 Java 和 LotusScript 類以及與 OpenAI 兼容的 REST API 支持。 ([en.wikipedia.org](https://en.wikipedia.org/wiki/HCL_Notes?utm_source=openai))

HCL Domino 2026 的這些更新旨在提升用戶體驗，增強安全性，並為開發者提供更強大的工具。
