## HCL Domino 2026: A New Era of Secure and Intelligent Collaboration

On March 19, 2026, HCL officially released HCL Domino 2026 (version 14.5.1), bringing a host of innovations and improvements to enterprise collaboration.

### Domino IQ: Private AI Solution

Domino 2026 introduces Domino IQ, an integrated private AI extension within the Domino platform. Utilizing Retrieval-Augmented Generation (RAG) technology, Domino IQ enables teams to query decades of organizational knowledge stored in Domino databases using natural language, all processed internally to ensure data privacy and security. ([hcl-software.com](https://www.hcl-software.com/domino/home?utm_source=openai))

### Enhanced User Experience

The new Notes client features cleaner layouts, modern navigation, and updated iconography, shaped by over 1,000 community votes to ensure the interface aligns with actual user workflows. ([hcl-software.com](https://www.hcl-software.com/domino/home?utm_source=openai))

### Strengthened Security and Compliance

Domino 2026 meets BSI (German Federal Office for Information Security) certification for information security and supports Security Event and Incident Management (SEIM) tools. Additionally, it complies with the European Accessibility Act, offering a more user-friendly web experience. ([hcltech.com](https://www.hcltech.com/press-releases/hclsoftware-launches-sovereign-ai-aimed-governments-and-regulated-organizations?utm_source=openai))

### Simplified Installation and Updates

The Windows installer for the new Notes client has been completely redesigned, with modules installed directly into the file system, similar to the fast Mac installation process, reducing installation time to approximately 38 seconds. ([madicon.de](https://www.madicon.de/blog/posts-blog/hcl-notesdomino-1451-is-here-all-new-features-at-a-glance/?utm_source=openai))

### Other Key Updates

- **REST API Enhancements**: Updated REST APIs and improved JSON parsing for LotusScript reduce the effort required to integrate Domino with modern development ecosystems and third-party tools.

- **Infrastructure Support**: Certified support for IBM i 7.6, Red Hat 10, and Windows Server 2025 ensures your infrastructure and Domino investments progress on the same timeline. ([hcl-software.com](https://www.hcl-software.com/domino/home?utm_source=openai))

These updates in HCL Domino 2026 aim to provide a more intelligent, secure, and efficient enterprise collaboration experience, enabling organizations to leverage AI technology to enhance productivity without compromising data privacy.
