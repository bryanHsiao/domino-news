HCL 近期發布了 HCL Nomad Web 1.0.20，為用戶帶來多項新功能和改進。以下是本次更新的主要亮點：

## Kiosk 模式

新版本引入了 Kiosk 模式，允許指定的資料庫在啟動時自動運行，適用於需要專注於特定應用的場景。

## 使用者介面改進

多個使用者介面控件已更新，以符合 HCL Enchanted 設計語言，包括：

- Rich Text Lite
- 時間和時區控件
- 顏色選擇器和顏色欄位
- 連結熱點

## 錯誤修復

本次更新還修復了多個已知問題，提升了應用的穩定性和性能。

更多詳細資訊，請參閱 [HCL Nomad for web browsers 1.0.x 使用者文件](https://help.hcl-software.com/nomad/1.0_web/PDF/nomad_web.pdf)。

如需下載最新版本，請訪問 [HCL 官方網站](https://developer.ds.hcl-software.com/t/hcl-nomad-for-web-browsers-and-hcl-nomad-server-on-domino-1-0-19-now-officially-released/172443)。
