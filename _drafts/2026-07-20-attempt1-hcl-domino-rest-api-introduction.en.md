## What is the HCL Domino REST API?

The HCL Domino REST API provides developers with a secure and modern way to access HCL Domino servers and databases through RESTful interfaces. This enables developers to use their preferred programming languages to extend and modernize existing Domino applications.

## Key Features

- **Security**: The Domino REST API inherits the security model of Notes and Domino, utilizing JSON Web Token (JWT) and Scopes for authentication, ensuring secure data access. ([opensource.hcltechsw.com](https://opensource.hcltechsw.com/Domino-rest-api/topicguides/introducingrestapi.html?utm_source=openai))

- **Modern Java API**: It employs industry-standard Java 8+ APIs, offering a more intuitive and user-friendly development experience. ([opensource.hcltechsw.com](https://opensource.hcltechsw.com/Domino-rest-api/topicguides/introducingrestapi.html?utm_source=openai))

- **Based on OpenAPI 3.0**: Provides comprehensive interactive documentation, facilitating developers' understanding and usage of the API. ([opensource.hcltechsw.com](https://opensource.hcltechsw.com/Domino-rest-api/references/openapidefinitions.html?utm_source=openai))

- **Extensible Architecture**: Supports multiple API versions and can be extended to add more functionalities as needed. ([opensource.hcltechsw.com](https://opensource.hcltechsw.com/Domino-rest-api/topicguides/introducingrestapi.html?utm_source=openai))

- **Web GUI Management**: Offers a web interface for managing schemas, scopes, and applications conveniently. ([opensource.hcltechsw.com](https://opensource.hcltechsw.com/Domino-rest-api/topicguides/introducingrestapi.html?utm_source=openai))

## Installation and Configuration

To get started with the Domino REST API, follow these steps:

1. **Download**: Obtain the Domino REST API installer or Docker image suitable for your platform from the HCL Software Portal. ([opensource.hcltechsw.com](https://opensource.hcltechsw.com/Domino-rest-api/tutorial/quickstart.html?utm_source=openai))

2. **Install and Run**: Follow the installation guide to set up and start the REST API on your Domino server or Notes client. ([opensource.hcltechsw.com](https://opensource.hcltechsw.com/Domino-rest-api/tutorial/quickstart.html?utm_source=openai))

3. **Learn the Tools**: Familiarize yourself with tools like Admin UI, Postman, and curl to interact with the API effectively. ([opensource.hcltechsw.com](https://opensource.hcltechsw.com/Domino-rest-api/tutorial/quickstart.html?utm_source=openai))

4. **Configure Databases**: Select the database to work with, create schemas, and define forms, views, and agents to expose via REST. ([opensource.hcltechsw.com](https://opensource.hcltechsw.com/Domino-rest-api/tutorial/quickstart.html?utm_source=openai))

5. **Test**: Use Swagger UI, Postman, or curl to test your API and ensure it functions correctly. ([opensource.hcltechsw.com](https://opensource.hcltechsw.com/Domino-rest-api/tutorial/quickstart.html?utm_source=openai))

## Conclusion

The HCL Domino REST API provides developers with a powerful and flexible tool to leverage modern development techniques in extending and modernizing Domino applications. By following the steps outlined above, you can quickly get started and take advantage of these new capabilities to enhance your application development experience.
