HCL Nomad is a client that allows users to access HCL Domino applications on mobile devices and web browsers. The application recently released version 1.0.54, primarily focusing on bug fixes to enhance stability and performance. You can download the latest version from the [App Store](https://apps.apple.com/us/app/hcl-nomad/id751904885).

According to the [HCL Nomad official documentation](https://help.hcl-software.com/nomad/welcome/index.html), Nomad supports multiple platforms, including iOS, Android, and web browsers. Each platform's latest version includes bug fixes and feature improvements aimed at enhancing user experience.

Additionally, as noted in a [panagenda blog post](https://www.panagenda.com/blog/customer-information-for-hcl-nomad-web-1-0-17/), Nomad Web version 1.0.17 introduced a refreshed user interface, improved login process, and multiple bug fixes, further enhancing platform stability and reliability.

Users are encouraged to update to the latest version to benefit from the most recent improvements and features.
