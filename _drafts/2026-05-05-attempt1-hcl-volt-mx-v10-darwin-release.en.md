HCL has recently released HCL Volt MX v10, codenamed 'Darwin', marking a significant upgrade to its low-code application development platform. This version introduces several new features aimed at enhancing development efficiency and application experience.

## Figma Design Import

The 'Figma Design Import' feature allows developers to convert designs from Figma directly into forms within Volt Iris, significantly reducing UI development time. This functionality leverages AI technology to swiftly transform design views into operational application interfaces.

## AI-Enhanced Development Assistant (AIAD)

Volt IQ now supports Retrieval Augmented Generation (RAG), providing more powerful conversational responses. Developers can ask the Volt IQ assistant questions, even generating code snippets, thereby accelerating the development process.

## Car Application Development Support

Volt Iris now supports car application development, enabling seamless integration with Apple CarPlay and Android Auto. This update opens up new possibilities for developers aiming to innovate in the automotive space.

## Other Enhancements

- **Passkey Support**: Introducing Passkey APIs based on the WebAuthn standard, enabling password-less authentication to enhance security and user experience.

- **iOS Live Activity**: Support for Live Activities in iOS 16.2+, allowing real-time updates on the Lock Screen and Dynamic Island, offering a less obtrusive alternative to traditional notifications.

- **Volt Iris UI/UX Revamp**: A redesign of the Volt Iris IDE to improve usability, accessibility, and aesthetics.

- **Modernized Widget Library**: Updated the default look and feel of the widget library to enhance usability and visual appeal.

- **Improved Error Reporting**: Added detailed and context-specific logging of errors in the Action Editor to facilitate easier troubleshooting and correction.

These new features and enhancements are designed to help developers more efficiently create multi-experience applications that meet modern user demands.
