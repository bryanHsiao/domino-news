HCL has announced the release of HCL Volt MX v10, codenamed 'Darwin', marking a significant upgrade to its multiexperience low-code application development platform. This version introduces several new features aimed at enhancing development efficiency and application diversity.

## Key Features

### Figma Design Import

Developers can now directly import Figma designs into Volt Iris, transforming designs into functional application interfaces, significantly reducing development time. For more details, refer to the [HCL Volt MX v10 'Darwin' blog post](https://www.hcl-software.com/blog/volt-mx/hcl-volt-mx-v10-darwin-from-figma-to-app-aiad-carplay-and-beyond).

### AI Augmented Development (AIAD)

Volt IQ now supports Retrieval Augmented Generation (RAG), providing smarter code generation and suggestions, assisting developers in writing code more efficiently. More information can be found in the [HCL Volt MX v10 'Darwin' blog post](https://www.hcl-software.com/blog/volt-mx/hcl-volt-mx-v10-darwin-from-figma-to-app-aiad-carplay-and-beyond).

### Car Application Development Support

Volt Iris now supports the development of applications for Apple CarPlay and Android Auto, expanding the reach of applications. Details are available in the [HCL Volt MX v10 'Darwin' blog post](https://www.hcl-software.com/blog/volt-mx/hcl-volt-mx-v10-darwin-from-figma-to-app-aiad-carplay-and-beyond).

### Passkey Support

Introducing Passkey APIs based on the WebAuthn standard, providing password-less authentication, enhancing application security and user experience. More information can be found in the [HCL Volt MX v10 'Darwin' blog post](https://www.hcl-software.com/blog/volt-mx/hcl-volt-mx-v10-darwin-from-figma-to-app-aiad-carplay-and-beyond).

## Additional Enhancements

- **Volt Iris UI/UX Revamp**: Redesigned Volt Iris IDE with improved usability and aesthetics.
- **Modernized Widget Library**: Updated the default widget library's look and feel to enhance usability.
- **Improved Error Reporting**: Added detailed error logging in the Action Editor to facilitate troubleshooting.

## Conclusion

HCL Volt MX v10 'Darwin' introduces AI-driven design conversion, car application development support, and password-less authentication, providing developers with powerful tools to efficiently develop multiexperience applications. For more information, refer to the [HCL Volt MX v10 'Darwin' blog post](https://www.hcl-software.com/blog/volt-mx/hcl-volt-mx-v10-darwin-from-figma-to-app-aiad-carplay-and-beyond) and the [latest release notes](https://help.hcl-software.com/voltmx/latest/VMX_release_notes.html).
