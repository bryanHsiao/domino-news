## Key Updates

### User Interface Enhancements

This release updates several controls to align with the HCL Enchanted design language, including:

- **Rich Text Lite**: Provides a more intuitive text editing experience.
- **Time and Time Zone Controls**: Improves the ease of selecting time and setting time zones.
- **Color Picker and Color Field**: Offers more flexible color selection capabilities.
- **Link Hotspot**: Enhances link management and navigation features.

### Bug Fixes

This version also addresses multiple known issues, further enhancing system stability and reliability. For a detailed list of fixes, please refer to the official release notes.
