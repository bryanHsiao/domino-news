HCL Domino 2026（版本 14.5.1）已於 2026 年 3 月 19 日正式發布，為企業提供了更現代化且安全的協作平台。以下是此版本的主要新功能和改進：

## 全新用戶界面

Domino 2026 引入了更清晰的佈局、現代化的導航和更新的圖標，這些改進是基於超過 1,000 名社群成員的投票結果，旨在使界面更符合用戶的工作習慣。 ([hcl-software.com](https://www.hcl-software.com/domino/home?utm_source=openai))

## 集成人工智慧功能

HCL Domino IQ 現在整合了檢索增強生成（Retrieval-Augmented Generation，RAG）技術，允許團隊使用自然語言安全地查詢存儲在 Domino 資料庫中的組織知識，且數據始終保持在內部控制之下。 ([hcl-software.com](https://www.hcl-software.com/domino/home?utm_source=openai))

## 增強的安全性

Domino 2026 通過 BSI IT-Grundschutz 認證，提供企業級的安全性，並引入了抗釣魚的 Passkey 認證，確保最高水平的數據隱私和安全性。 ([hcl-software.com](https://www.hcl-software.com/domino/home?utm_source=openai))

## 改進的 REST API 和 JSON 解析

更新的 REST API 和改進的 LotusScript JSON 解析功能，減少了將 Domino 與現代開發生態系統和第三方工具集成所需的工作量。 ([hcl-software.com](https://www.hcl-software.com/domino/home?utm_source=openai))

## 支援最新的操作系統

Domino 2026 現在支援 IBM i 7.6、Red Hat 10 和 Windows Server 2025，確保您的基礎設施和 Domino 投資能夠同步升級。 ([hcl-software.com](https://www.hcl-software.com/domino/home?utm_source=openai))

## 下載與升級

Domino 2026 的安裝包現已可通過 My HCLSoftware 下載門戶獲取。建議現有用戶盡快升級，以利用新功能和增強的安全性。 ([developer.ds.hcl-software.com](https://developer.ds.hcl-software.com/t/hcl-domino-2026-14-5-1-is-here/172452?utm_source=openai))
