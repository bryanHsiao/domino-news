HCL Nomad version 1.0.19 has been officially released, bringing a host of enhancements and fixes aimed at improving usability and stability. Here are the key highlights of this update:

## User Interface Enhancements

Several controls have been refreshed to align with the HCL Enchanted design language, including:

- **Rich Text Lite**: Offers a more intuitive text editing experience.
- **Time and Time Zone Controls**: Improved usability for time selection and time zone settings.
- **Color Picker and Color Field**: Provides more flexible color selection options.
- **Link Hotspot**: Enhances link management and navigation capabilities.

These updates aim to deliver a more consistent and modern user experience.

## Bug Fixes

This release also addresses multiple known issues, further enhancing platform stability and performance. For a detailed list of fixes, refer to the [HCL Nomad 1.0.19 Release Announcement](https://developer.ds.hcl-software.com/t/hcl-nomad-for-web-browsers-and-hcl-nomad-server-on-domino-1-0-19-now-officially-released/172443).

## Download and Installation

HCL Nomad 1.0.19 is now available for download on the [My HCLSoftware Portal](https://developer.ds.hcl-software.com/t/hcl-nomad-for-web-browsers-and-hcl-nomad-server-on-domino-1-0-19-now-officially-released/172443). Users are encouraged to upgrade promptly to take advantage of the latest features and improvements.

For more information, please refer to the [HCL Nomad User Documentation](https://help.hcl-software.com/nomad/1.0_web/PDF/nomad_web.pdf).
