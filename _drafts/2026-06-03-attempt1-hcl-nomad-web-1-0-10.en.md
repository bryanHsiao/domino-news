On December 13, 2023, HCL released Nomad Web 1.0.10, bringing several new features and improvements to enhance user experience.

## Integrated Designer

This version introduces an integrated designer, allowing users to create or edit Notes databases directly within the browser. By selecting the "Designer" option from the application's context menu, users can enter design mode. Note that developer rights on the database are required to access this feature. For more details, refer to [HCL Nomad for web browsers 1.0.10](https://alichtenberg.cz/hcl-nomad-for-web-browsers-1-0-10/).

## Offline Mail Sending

Version 1.0.10 supports offline mail sending. Users can compose and send emails without an internet connection; the emails are stored in the local mail.box and automatically sent once the connection is restored. Detailed information is available in [Customer Information for HCL Nomad Web 1.0.10](https://www.panagenda.com/blog/customer-information-for-hcl-nomad-web-1-0-10/).

## HTTP Connection Support

Nomad server now supports HTTP connections in addition to the previously supported HTTPS connections, providing more connection options. This is particularly useful for users needing to connect to reverse proxies via the HTTP protocol. More information can be found in the [HCL Nomad for web browsers User Documentation](https://help.hcl-software.com/nomad/1.0_web/PDF/nomad_web.pdf).

## Additional Features and Fixes

- **Optional HTTP Headers Support**: Nomad server now supports adding optional HTTP headers to the response.
- **Let's Encrypt Integration**: Simplifies SSL/TLS certificate management, enhancing security.
- **Language Support**: Added support for multiple languages, improving the experience for global users.
- **Domino Restyle Updates**: Improved the visual style of applications, providing a more modern appearance.

Additionally, version 1.0.10 addresses 16 known issues, further enhancing system stability and performance. For a complete list of updates and fixes, refer to [Customer Information for HCL Nomad Web 1.0.10](https://www.panagenda.com/blog/customer-information-for-hcl-nomad-web-1-0-10/).

## Conclusion

The release of HCL Nomad Web 1.0.10 brings several practical new features and improvements, further enhancing the experience of using Notes applications in the browser. Users are encouraged to upgrade to this version promptly to benefit from the latest features and fixes.
