HCL Nomad version 1.0.19 has been officially released, bringing a series of UI enhancements and bug fixes aimed at improving the overall user experience. Here are the key highlights of this update:

## UI Enhancements

Several controls have been refreshed to align with the HCL Enchanted design language, including:

- **Rich Text Lite**: Offers a more intuitive text editing experience.
- **Time and Time Zone Controls**: Enhances the ease of selecting time and setting time zones.
- **Color Picker and Color Field**: Provides more flexible color selection capabilities.
- **Link Hotspot**: Improves link management and navigation functionalities.

## Bug Fixes

This release also addresses multiple known issues, enhancing the application's stability and performance.

For more information, please refer to the [HCL Nomad 1.0.19 Release Announcement](https://developer.ds.hcl-software.com/t/hcl-nomad-for-web-browsers-and-hcl-nomad-server-on-domino-1-0-19-now-officially-released/172443) and the [HCL Nomad What's New Documentation](https://help.hcl-software.com/nomad/1.0_web/nomad_web_new.html).

Users are encouraged to update to the latest version promptly to benefit from these improvements.
