HCL has recently released HCL Nomad Web 1.0.20, bringing several new features and improvements to enhance user experience. Here are the key highlights of this update:

## Kiosk Mode

The new version introduces Kiosk Mode, allowing a specific database to automatically run on startup. This feature is particularly useful for scenarios requiring focus on a particular application.

## User Interface Enhancements

Multiple user interface controls have been updated to align with the HCL Enchanted design language, including:

- Rich Text Lite
- Time and time zone controls
- Color picker and color field
- Link hotspot

## Bug Fixes

This update also addresses several known issues, improving the application's stability and performance.

For more detailed information, refer to the [HCL Nomad for web browsers 1.0.x User Documentation](https://help.hcl-software.com/nomad/1.0_web/PDF/nomad_web.pdf).
