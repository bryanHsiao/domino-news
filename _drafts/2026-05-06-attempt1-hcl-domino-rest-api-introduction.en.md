## What is the HCL Domino REST API?

The HCL Domino REST API offers a secure RESTful interface to HCL Domino servers, allowing developers to access Domino databases using various programming languages. This enables the use of modern development tools and frameworks to extend and modernize existing Domino applications.

## Key Features

- **Security**: Inherits the security model of Notes and Domino, utilizing JSON Web Token (JWT) and Scopes for authentication, ensuring secure data access.

- **Modern Java API**: Employs industry-standard Java 8+ APIs, providing a more intuitive and efficient development experience.

- **Public API Based on OpenAPI 3.0**: Adheres to OpenAPI 3.0 standards, offering interactive API documentation for ease of understanding and use.

- **Extensible Architecture**: Supports multiple API versions and can be extended to add more functionalities as needed.

- **Web GUI Management**: Provides a web interface for managing schemas, scopes, and applications.

## Installation and Configuration

To get started with the Domino REST API, follow these steps:

1. **Download the Installer**: Obtain the latest Domino REST API installer or Docker image from the [My HCLSoftware Portal](https://opensource.hcltechsw.com/Domino-rest-api/tutorial/quickstart.html).

2. **Install and Run**: Follow the [installation and configuration guide](https://opensource.hcltechsw.com/Domino-rest-api/tutorial/quickstart.html) to install and start the REST API on your Domino server or Notes client.

3. **Learn to Use the Tools**: Familiarize yourself with tools like Admin UI, Postman, and curl for testing and managing the API.

4. **Configure Your Database**: Choose a database, create a schema defining the forms and views to expose via REST, and create a scope to activate API access.
