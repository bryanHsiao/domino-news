HCL Nomad version 1.0.19 was officially released on March 10, 2026, and is now available for download from the [My HCLSoftware Portal](https://developer.ds.hcl-software.com/t/hcl-nomad-for-web-browsers-and-hcl-nomad-server-on-domino-1-0-19-now-officially-released/172443). This release includes several user interface (UI) enhancements and bug fixes aimed at improving the overall user experience.

## Key Updates

### UI Enhancements

Several controls have been refreshed to align with the HCL Enchanted design language, including:

- Rich text lite
- Time and time zone controls
- Color picker and color field
- Link hotspot

These updates provide a more consistent and intuitive application appearance and operation.

### Bug Fixes

This release also addresses multiple known issues, enhancing the application's stability and reliability.

## Download and Additional Information

HCL Nomad 1.0.19 is now available for download from the [My HCLSoftware Portal](https://developer.ds.hcl-software.com/t/hcl-nomad-for-web-browsers-and-hcl-nomad-server-on-domino-1-0-19-now-officially-released/172443). For more detailed information, please refer to the [HCL Nomad product documentation](https://help.hcl-software.com/nomad/1.0_web/nomad_web_new.html).

Users are encouraged to upgrade to this version promptly to take advantage of the latest features and improvements.
