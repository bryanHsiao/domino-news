HCL Domino 2026 (version 14.5.1) was officially released on March 19, 2026, bringing a host of new features and enhancements for Notes clients, Domino servers, and developers.

## New Features in the Notes Client

- **Redesigned Windows Installer**: The installation process has been overhauled, with modules placed directly in the file system, similar to the streamlined Mac installation process. ([madicon.de](https://www.madicon.de/blog/posts-blog/hcl-notesdomino-1451-is-here-all-new-features-at-a-glance/?utm_source=openai))

- **User Interface Improvements**: A smoother UI, improved workspace, and enhanced calendar functionality contribute to a better overall user experience. ([developer.ds.hcl-software.com](https://developer.ds.hcl-software.com/t/hcl-domino-2026-14-5-1-is-here/172452?utm_source=openai))

## Enhancements in the Domino Server

- **Performance Boosts**: Enhanced performance and stronger security measures provide a more stable server environment. ([developer.ds.hcl-software.com](https://developer.ds.hcl-software.com/t/hcl-domino-2026-14-5-1-is-here/172452?utm_source=openai))

- **Improved Admin Tools**: Upgraded administration and configuration tools enable administrators to manage servers more effectively. ([developer.ds.hcl-software.com](https://developer.ds.hcl-software.com/t/hcl-domino-2026-14-5-1-is-here/172452?utm_source=openai))

- **Multi-File Upload Support in XPages**: XPages now supports multi-file uploads, increasing application flexibility. ([developer.ds.hcl-software.com](https://developer.ds.hcl-software.com/t/hcl-domino-2026-14-5-1-is-here/172452?utm_source=openai))

## Updates for Developers

- **Domino REST API Enhancements**: The latest version of the Domino REST API introduces new features and improvements, empowering developers with more capabilities. ([opensource.hcltechsw.com](https://opensource.hcltechsw.com/Domino-rest-api/whatsnew/index.html?utm_source=openai))

- **Introduction of Domino IQ**: The release introduces secure AI features (Domino IQ), allowing for local execution of large language models and providing Java and LotusScript classes, as well as OpenAI-compatible REST API support. ([en.wikipedia.org](https://en.wikipedia.org/wiki/HCL_Notes?utm_source=openai))

These updates in HCL Domino 2026 aim to enhance user experience, bolster security, and provide developers with more powerful tools.
