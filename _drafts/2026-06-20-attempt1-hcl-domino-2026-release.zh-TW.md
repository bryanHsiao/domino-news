## HCL Domino 2026：全新功能與強化安全的企業協作平台

HCL 於 2026 年 3 月 19 日正式發布了 HCL Domino 2026（版本 14.5.1），為企業協作平台帶來了多項創新與改進。

### Domino IQ：私有 AI 解決方案

Domino 2026 引入了 Domino IQ，這是一個整合於 Domino 平台的私有 AI 擴充功能。Domino IQ 採用檢索增強生成（Retrieval-Augmented Generation，RAG）技術，允許團隊使用自然語言查詢儲存在 Domino 資料庫中的多年組織知識，且所有資料均在企業內部處理，確保資料隱私與安全。 ([hcl-software.com](https://www.hcl-software.com/domino/home?utm_source=openai))

### 使用者體驗的提升

新版 Notes 客戶端採用了更清晰的佈局、現代化的導航以及更新的圖示，這些改進是基於超過 1,000 名社群成員的投票結果，確保介面更符合使用者的實際需求。 ([hcl-software.com](https://www.hcl-software.com/domino/home?utm_source=openai))

### 強化的安全性與合規性

Domino 2026 符合 BSI（德國聯邦資訊安全辦公室）的資訊安全認證，並支援安全事件與事件管理（SEIM）工具。此外，該版本符合歐洲無障礙法案，提供更友善的網頁使用者體驗。 ([hcltech.com](https://www.hcltech.com/press-releases/hclsoftware-launches-sovereign-ai-aimed-governments-and-regulated-organizations?utm_source=openai))

### 簡化的安裝與更新

新版 Notes 客戶端的 Windows 安裝程式經過全面重新設計，模組直接安裝到檔案系統中，類似於 Mac 客戶端的快速安裝方式，安裝時間約為 38 秒。 ([madicon.de](https://www.madicon.de/blog/posts-blog/hcl-notesdomino-1451-is-here-all-new-features-at-a-glance/?utm_source=openai))

### 其他重要更新

- **REST API 更新**：改進的 REST API 和增強的 LotusScript JSON 解析功能，減少了將 Domino 與現代開發生態系統和第三方工具整合所需的工作量。

- **基礎設施支援**：支援 IBM i 7.6、Red Hat 10 和 Windows Server 2025，確保您的基礎設施與 Domino 投資同步發展。 ([hcl-software.com](https://www.hcl-software.com/domino/home?utm_source=openai))

HCL Domino 2026 的這些更新旨在提供更智能、更安全且更高效的企業協作體驗，幫助組織在不影響資料隱私的前提下，充分利用 AI 技術提升生產力。
